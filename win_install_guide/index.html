Windows Build Guide¶

Author:

Christoph Schmidt-Hieber

Release:

dev

Date:

20 March, 2026

Current Windows builds use Visual Studio, CMake presets, vcpkg, and CPack. The supported path is the modern CMake workflow; the historical Visual C++ Express material is no longer part of the primary build guide.
